##Getting Started

###.env

MONGO_URI=your_mongodb_url
JWT_SECRET=your_jwt_secret_key

###Installing

### **Frontend**

cd Frontend
npm install
npm run dev

### **Backend**
cd Backend
npm install
add env file
npm start
